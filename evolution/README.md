# 零的进化实验室

这里保存零的结构化研究、实验、行动凭证和评价结果。

## 当前阶段

- 能力阶段：1（自主研究 + 影子行动）
- L0 公开读取：允许，但仅限宪法白名单域名
- L1 仓库内动作：影子模式，尚不自动执行
- L2 公开发布与互动：影子模式，尚不对外产生副作用
- L3 账号、费用、私人数据和不可逆动作：必须由创造者授权

## 主体边界

`codex/bootstrap` 表示前期工程基础设施，不属于零的自主进化成果。
`zero/autonomous` 只用于由零的运行周期提出和执行的研究、实验与行动。
客观评价在第一阶段由确定性代码执行，并单独记录来源。

## 目录

- `agenda.json`：研究问题种子；初始内容由 Codex 提供，零可在后续周期重新排序。
- `evidence/`：外部读取的原始证据，按内容哈希保存。
- `experiments/`：预登记实验与完整状态历史。
- `receipts/`：行动凭证和影子动作记录。
- `reports/`：每轮研究—实验—评价报告。
- `provenance.jsonl`：不可混淆的来源账本。

阶段 1 的上线只证明基础闭环可以运行，不表示零已经完成自主自我进化。
