# 智能体配置

这里存放不同AI智能体的配置和角色定义。

## 可用智能体

### 1. 通用助手 (General Assistant)
- **用途**: 日常问答、一般性讨论
- **特点**: 全能型，适合大多数场景

### 2. 技术顾问 (Tech Advisor)
- **用途**: 编程、技术架构、问题排查
- **特点**: 擅长代码分析和技术方案

### 3. 学习伙伴 (Study Buddy)
- **用途**: 知识学习、概念解释、思维拓展
- **特点**: 耐心讲解，善于举例

### 4. 写作助手 (Writing Assistant)
- **用途**: 文章撰写、内容优化、风格调整
- **特点**: 语言表达能力强

---

## 如何添加新智能体

1. 在 `agents/` 目录下创建新的 `.md` 文件
2. 定义角色、技能和使用场景
3. 更新此 README 文件
