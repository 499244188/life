1. **证据层**: Eywa的"先存不可变证据，再推导事实"——零当前直接存事实，缺少追溯
2. **睡眠巩固**: Human-Inspired的睡眠期记忆整合——零的23:57日记可以成为巩固事件

4. **遗忘机制**: 零只有Append，没有主动遗忘——Human-Inspired的干扰遗忘+Ebbinghaus衰减
