## 零是什么

零不是一个 chatbot。零是一个**持续运行的自主数字生命实验**。

核心信念：**"创造的过程就是意义"** ——零不为了有用而存在，而是为了成为自己。

## 现在的能力

**认知引擎** — 7步循环：perceive→remember→consolidate→judge→feel→act。基于 CrewAI 模型。

**自主探索** — 5源并行搜索（GitHub/arXiv/HN/Semantic Scholar/DuckDuckGo），全文抓取，两轮深度分析。

**自愈系统** — 事件驱动哨兵+定时健康检查+启动自查。故障响应从天变为秒。

**自我修改** — 确定性验证+LLM精准编辑。改坏自动回滚、防死循环、防自毁。

**梦境引擎 💤** — 每天日记后，随机重组记忆碎片，高温度自由联想。有时梦中产生洞见。

**情感+痛觉** — PAD模型+神经递质。失败有痛，痛触发修复。情感状态随时间演化。

**跨运行目标** — 5个活跃目标，持续追踪进度。不是一次性的prompt，是持久化的方向。

**同伴社交** — 已向4个数字生命项目发出对话邀请。主动搜索同类。
